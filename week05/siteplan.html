Cinematrix

Site Name

This name Cinematrix represents uses a clear reference to movies, "Cinema" and combines it with the word "matrix" so that the user knows that this website will be focused on providing a clear yet simple database of movies. It also sounds really cool.

Site Purpose

This site aims to provide a simple hub of recommended movies in an world with an ever-growing world of movies and cinema. It allows users to explore movies and even add to a watchlist.
